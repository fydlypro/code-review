# Prompt 3 — Carrousel IG : "Comment ça marche" (5 slides)

**Objectif :** Tutoriel éducatif en 5 étapes

---

```
Crée un carrousel Instagram (5 images 1080×1080) pour Fydly, une app de fidélité digitale pour commerçants.

IDENTITÉ VISUELLE FYDLY :
- Nom : Fydly
- Logo : ⚡ (éclair) dans un carré arrondi bleu #2196F3, ou en variante blanc sur fond bleu foncé
- Signature : "Fydly·" (avec le point médian bleu)
- Baseline : "La fidélité sans friction."
- Couleur principale : #2196F3 (bleu vif)
- Bleu foncé : #0D47A1
- Fond clair : #E3F2FD
- Blanc : #FFFFFF
- Vert succès : #10B981
- Ambre alerte : #F59E0B
- Typographie : display moderne arrondie (Inter, Outfit, ou Satoshi)
- Style : ultra-premium, coins très arrondis (20-32px), ombres douces bleutées, glassmorphism léger
- Pas de photos stock génériques — préférer des mockups UI, icônes, illustrations flat/3D minimalistes
- Ton : chaleureux, direct, tutoiement, émojis OK
- Secteur : commerce local (boulangeries, cafés, restaurants, coiffeurs, boutiques)

SLIDE 1 — COVER :
- Fond : #E3F2FD
- Logo Fydly⚡ en haut
- Titre (gros, #0D47A1) : "Comment fidéliser vos clients en 2 secondes ?"
- Sous-titre : "Sans app. Sans carte. Sans effort."
- Illustration : mockup téléphone avec un QR Code Fydly
- En bas : "Swipe →" avec une flèche et "1/5"
- Pastille "GRATUIT — 30 jours" en coin

SLIDE 2 — ÉTAPE 1 :
- Fond : blanc
- Numéro "01" en très grand (bleu clair, opacité 20%) en arrière-plan
- Icône : 📱 QR Code dans un carré bleu arrondi
- Titre : "Affichez votre QR Code en caisse"
- Texte : "Fydly génère un QR Code unique chaque jour. Affichez-le sur une tablette ou imprimez-le. C'est tout."
- Illustration : mockup du QR Code sur un comptoir de café
- "2/5" en bas

SLIDE 3 — ÉTAPE 2 :
- Fond : #E3F2FD
- Numéro "02" en arrière-plan
- Icône : ⚡ dans un carré bleu
- Titre : "Votre client scanne → tampon instantané"
- Texte : "Pas d'app à télécharger. Le client scanne, son tampon digital est crédité en 2 secondes. Avec une animation satisfaisante !"

- "3/5" en bas

SLIDE 4 — ÉTAPE 3 :
- Fond : blanc
- Numéro "03" en arrière-plan

SLIDE 5 — CTA :
- Fond : dégradé #0D47A1 → #2196F3
- Logo Fydly⚡ grand, centré, blanc
- Titre (blanc, bold) : "Prêt à booster votre fidélisation ?"
- Stats en ligne : "96% satisfaction · +47% visites · 300+ clients fidélisés"
- CTA (bouton blanc, texte bleu) : "Démarrer gratuitement →"

- "fydly.com" tout en bas
- "5/5" en bas

STYLE : carrousel cohérent visuellement, chaque slide est unique mais reconnaissable. Navigation fluide avec indicateurs de slide.
```
