# Prompt 1 — Story IG/TikTok : "Arrête les cartes papier"

**Format :** Story Instagram / TikTok (1080×1920)
**Objectif :** Accroche impactante, avant/après papier → digital

---

```
Crée une story Instagram (1080×1920) pour Fydly, une app de fidélité digitale pour commerçants.

IDENTITÉ VISUELLE FYDLY :
- Nom : Fydly
- Logo : ⚡ (éclair) dans un carré arrondi bleu #2196F3, ou en variante blanc sur fond bleu foncé
- Signature : "Fydly·" (avec le point médian bleu)
- Baseline : "La fidélité sans friction."
- Couleur principale : #2196F3 (bleu vif)
- Bleu foncé : #0D47A1
- Fond clair : #E3F2FD
- Blanc : #FFFFFF
- Vert succès : #10B981
- Ambre alerte : #F59E0B
- Typographie : display moderne arrondie (Inter, Outfit, ou Satoshi)
- Style : ultra-premium, coins très arrondis (20-32px), ombres douces bleutées, glassmorphism léger
- Pas de photos stock génériques — préférer des mockups UI, icônes, illustrations flat/3D minimalistes
- Ton : chaleureux, direct, tutoiement, émojis OK
- Secteur : commerce local (boulangeries, cafés, restaurants, coiffeurs, boutiques)

CONTENU :
- Fond : dégradé vertical de #0D47A1 (haut) vers #2196F3 (bas)
- En haut : petit logo Fydly⚡ en blanc
- Texte principal (gros, centré, blanc, font display bold) :
  "Arrête les cartes tampons en carton. 🗑️"
  Ligne 2 (plus petit, blanc/80%) : "Tes clients les perdent. Toujours."
- Au milieu : mockup d'une carte de fidélité papier barrée/froissée à gauche, avec une flèche → vers un mockup de téléphone montrant une carte Fydly digitale (grille de tampons ⚡ bleus, 7/10 remplis, barre de progression, récompense "1 café offert")
- En bas : 
  CTA : bouton blanc arrondi "Essai gratuit — 30 jours" 
  Sous le bouton : "Sans carte bancaire · fydly.com" en blanc/60%

STYLE : contrasté, impactant, minimaliste. L'accent est sur le visuel avant/après (papier → digital).
```
