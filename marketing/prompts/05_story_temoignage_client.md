# Prompt 5 — Story IG : Témoignage Client

**Format :** Story Instagram (1080×1920)
**Objectif :** Preuve sociale via témoignage authentique

---

```
Crée une story Instagram (1080×1920) type témoignage pour Fydly.

IDENTITÉ VISUELLE FYDLY :
- Nom : Fydly
- Logo : ⚡ (éclair) dans un carré arrondi bleu #2196F3, ou en variante blanc sur fond bleu foncé
- Signature : "Fydly·" (avec le point médian bleu)
- Baseline : "La fidélité sans friction."
- Couleur principale : #2196F3 (bleu vif)
- Bleu foncé : #0D47A1
- Fond clair : #E3F2FD
- Blanc : #FFFFFF
- Vert succès : #10B981
- Ambre alerte : #F59E0B
- Typographie : display moderne arrondie (Inter, Outfit, ou Satoshi)
- Style : ultra-premium, coins très arrondis (20-32px), ombres douces bleutées, glassmorphism léger
- Pas de photos stock génériques — préférer des mockups UI, icônes, illustrations flat/3D minimalistes
- Ton : chaleureux, direct, tutoiement, émojis OK
- Secteur : commerce local (boulangeries, cafés, restaurants, coiffeurs, boutiques)

CONTENU :
- Fond : blanc
- En haut : logo Fydly⚡ + 5 étoiles ⭐ dorées
- Guillemets ouvrants géants (bleu #2196F3, opacité 15%) en arrière-plan
- Citation (texte principal, font display, #0D47A1, taille moyenne-grande) :
  "Depuis Fydly, mes clients reviennent 40% plus souvent. Et ils adorent l'animation quand ils gagnent un tampon — ça fait vraiment la différence."
- Photo/avatar : cercle bleu #2196F3 avec initiale "M" en blanc (pas de vraie photo)
- Nom : "Marie Fontaine" (bold, #0D47A1)
- Sous-titre : "Propriétaire — Boulangerie Fontaine, Lyon" (bleu #2196F3, petit)
- En bas :
  - Ligne séparatrice fine bleu clair
  - CTA : "Rejoignez Marie → fydly.com" 
  - "Essai gratuit — 30 jours" en petit

STYLE : authentique, inspirant, minimaliste. Le témoignage est le héros du visuel.
```
